Este directorio contiene los conjuntos de datos utilizados para el entrenamiento, validación y testeo.

##  Estructura del Dataset
El dataset principal es una simulación sintética de **8,758 registros horarios** que replican el comportamiento termodinámico de una planta industrial real.

##  Nota sobre el Origen
Los datos son sintéticos realistas para permitir la validación de la arquitectura del modelo sin comprometer información sensible de planta, manteniendo la estructura necesaria para una futura ingesta de datos reales vía SCADA.
